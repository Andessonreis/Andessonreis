<div align="center">
  <h2 align="center"> ⚙️ Tecnologias e Habilidades </h2><br>  
</div>

### **Engenharia de Dados:**
- **Tecnologias de Manipulação:** Python (manipulação de dados, script e automação), SQL (consulta avançada e otimização de banco de dados).
- **Big Data e Ferramentas ETL/ELT:** Proficiência em Apache Spark, Hadoop, Apache Airflow, Apache Nifi e Talend.
- **Arquitetura de Dados:** Design e manutenção de data warehouses e data lakes.
- **Cloud e Armazenamento:** Experiência em AWS, GCP e Azure para gerenciamento de dados.
- **Visualização de Dados:** Proficiente em ferramentas como Tableau, Power BI e matplotlib.
  
### **Desenvolvimento de Backend:**
- **Construção de API e Lógica do Servidor:** Utilização de Java e Spring para construção de API e lógica do servidor.
- **Tecnologias e Frameworks Web:** Domínio em Bootstrap, jQuery, JavaScript, HTML5 e CSS3 para desenvolvimento web.
- **Soluções Robustas no Backend:** Desenvolvimento e implementação de soluções eficientes no backend.
- **Versatilidade em Linguagens:** Proficiência em PyTorch, Tidyverse, C, C++ e práticas de versionamento Git.
